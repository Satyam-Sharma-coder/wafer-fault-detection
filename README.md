# Semiconductor Wafer Fault Detection

An autoencoder-based fault detection system for semiconductor wafers using normal behaviour modelling, reconstruction error analysis, and statistical thresholding.

## Features

- Missing value preprocessing
- Autoencoder-based anomaly detection
- Reconstruction error thresholding
- Severity classification (Low / Medium / High)
- Confusion matrix & metrics
- Flask web UI
- Production-ready structure
- Docker & Render deployment ready

## Tech Stack

- Python 3.11
- Flask
- TensorFlow / Keras
- Scikit-learn
- Pandas, NumPy
- Bootstrap
- Matplotlib
- Gunicorn
